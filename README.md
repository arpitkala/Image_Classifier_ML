# 🏅 Sports Classifier AI

> A production-ready deep learning image classification project that identifies **100+ sports** from photos using a fine-tuned MobileNetV2 model — complete with a Google Colab training notebook and a Streamlit web application.

---

## 📸 Demo

| Upload | Predict | Results |
|--------|---------|---------|
| Drop any sports photo | Click Classify | Get sport name + confidence + Top-3 |

## 🚀 Quick Start

### 1. Clone / Download the project

```bash
git clone https://github.com/your-username/sports-classifier.git
cd sports-classifier
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

> Python **3.9 – 3.11** recommended. GPU optional but speeds up training.

---

## 📓 Part A: Train the Model (Notebook)

### Option 1 — Google Colab (Recommended)

1. Open [Google Colab](https://colab.research.google.com/).
2. Upload `notebook.ipynb`.
3. Go to **Runtime → Change runtime type → GPU**.
4. Upload your `kaggle.json` API key when prompted.
5. Click **Runtime → Run all**.
6. After training completes, download `model/model.h5` and `model/class_names.json`.

### Option 2 — Local Jupyter

```bash
pip install jupyter
jupyter notebook notebook.ipynb
```

Make sure your `~/.kaggle/kaggle.json` is configured before running the download cell.

#### Getting a Kaggle API key:
1. Go to [kaggle.com](https://www.kaggle.com/) → Account → API → **Create New API Token**.
2. Save the downloaded `kaggle.json` to `~/.kaggle/kaggle.json`.
3. On Linux/Mac: `chmod 600 ~/.kaggle/kaggle.json`

---

## 💻 Part B: Run the Web App

> ⚠️ You must have `model/model.h5` and `model/class_names.json` before running the app.

```bash
streamlit run app.py
```

Open your browser at **http://localhost:8501**.

### App Features
- 📤 Upload any JPG / PNG / WEBP image
- 🔍 One-click prediction
- 🏆 Predicted sport with confidence percentage
- 📊 Top-3 predictions with visual bars
- 🔬 Full probability distribution (expandable)
- ⚡ Inference time display

## 🧠 Model Architecture

| Component | Detail |
|-----------|--------|
| Base model | MobileNetV2 (ImageNet pretrained) |
| Input size | 224 × 224 × 3 |
| Classifier head | GAP → Dense(256) → BN → Dropout(0.4) → Softmax |
| Fine-tuning | Top 30 layers of MobileNetV2 unfrozen |
| Loss | Categorical Cross-Entropy |
| Optimizer | Adam (lr=1e-3 → 1e-5 fine-tune) |
| Metrics | Top-1 Accuracy, Top-3 Accuracy |

---

## 📊 Training Pipeline

```
Dataset Download (Kaggle API)
        ↓
Data Preprocessing + Augmentation
  (rotation, zoom, flip, brightness)
        ↓
Model 1: Custom CNN (4 Conv blocks)
        ↓
Model 2: MobileNetV2 Transfer Learning (frozen base)
        ↓
Fine-tuning (unfreeze top 30 layers, lr=1e-5)
        ↓
Evaluation (Accuracy, Top-3, Confusion Matrix)
        ↓
Save model.h5 + class_names.json
```

## 📦 Dataset

- **Source:** [Kaggle — Sports Classification by gpiosenka](https://www.kaggle.com/datasets/gpiosenka/sports-classification)
- **Classes:** ~100 sports
- **Images:** ~14,000+ (train / valid / test split provided)
- **Format:** RGB JPEG, various resolutions

## 📈 Expected Results

| Model | Val Top-1 | Val Top-3 |
|-------|-----------|-----------|
| Custom CNN | ~75% | ~90% |
| MobileNetV2 (frozen) | ~88% | ~97% |
| MobileNetV2 (fine-tuned) | **~93%** | **~99%** |

*Results may vary slightly depending on hardware, TF version, and random seed.*
